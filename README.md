# Fitness & Health Care App

A modern, interactive fitness and health tracking application built with React, Tailwind CSS, and Vite.

## Features

- **Splash Screen**: Beautiful animated welcome screen with gradient background and pulsing rings
- **Login Screen**: User and Admin login options
- **Dashboard**: Real-time metrics display including:
  - Steps tracking
  - Sleep monitoring
  - Calorie tracking
  - Health Score
  - Daily insights (water intake, active time, mood)
  
- **Fitness Section**: Workout plans with progress tracking
- **Wellness Section**: Stress level, breathing exercises, and daily motivation
- **Nutrition Section**: Diet tips and water intake tracker
- **Progress Section**: Weekly activity trends and averages

## Tech Stack

- **React 18.2** - UI framework
- **Vite** - Fast build tool
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Icon library

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Usage

1. Start the development server
2. View the splash screen (2.8 seconds)
3. Select User or Admin login
4. Browse through different sections:
   - Dashboard: View your metrics
   - Fitness: Plan and track workouts
   - Wellness: Manage stress and well-being
   - Nutrition: Track diet and hydration
   - Progress: View activity trends
